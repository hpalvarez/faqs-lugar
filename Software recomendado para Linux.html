Bienvenidos a la página de soft recomendado del L.U.G.Ar

En esta página del LUGAr conoceran sofware de 1ra. para Linux. Todo el mundo cree que hay poco soft para Linux... les recomiendo que vean esto y se den despues una vueltita por Freshmeat donde todos los dias se anuncian los lanzamientos de soft para Linux (hay como 20 por dia!).
Aquí yo hice una selección de algunos que probe y estan muy buenos. Hice una escala para clasificarlos en "pinguinos": cuantos mas pinguinos mejor el programa (de 1 a 10 pinguinos).
Tambien incluí en el si necesita alguna libreria rara que no suela venir incluida en las distribuciones mas populares. Y por supuesto la home page correspondiente, para obtener más información sobre el programa y bajarlo si les gusta. El número de versión del programa también lo puse en los casos que lo se.

Una ultima cosa, los programas a continuacion se pueden encontrar -la mayoria- ya compilados y empaquetados en .deb o .rpm, depende de su distribucion.

Licq

  • Clon del ICQ para Linux.
  • Ultima versión: 0.61 (vers. estable) 0.70d (vers. de desarrollo)
  • Implementa casi todas las caracteristicas del ICQ original: chat, mensajes, URLs ...
  • 8,5 pinguinos
  • Necesita: Libreria Qt 1.44 ( http://www.troll.no )
  • Home page: http://licq.wibble.net

Linux-logo

  • Muestra un logo de Linux cuando arranca el sistema.
  • Ultima versión: 3.0
  • 7 pinguinos
  • Home page: http://www.glue.umd.edu/~weave

mpg-123

  • Reproductor de archivos MPEG de audio en la consola (modo texto)
  • MUY rapido, y muchos front-ends disponibles.
  • Ultima versión: 0.59q
  • 8 pinguinos.
  • Home page: http://mpg.123.org/

BitchX

  • Cliente de IRC en modo texto.
  • Muy bueno. Escribe en colores, tiene como 200 comandos.
  • Ultima versión: 75p3
  • 8 pinguinos.
  • Home page: http://www.bitchx.com

Mutt

  • Cliente de e-mail para consola.
  • Buenismo, muy potente, soporta muchos estandares.
  • Ultima versión: 0.95.6
  • 9 pinguinos.
  • Home page: http://www.mutt.org

X11Amp

  • Clon del Winamp para Linux
  • Igualito al Winamp, hasta soporta las skins!
  • Ultima versión: 0.9
  • 8 pinguinos
  • Home page: http://www.x11amp.org/

XFMail

  • Excelente cliente de e-mail.
  • Soporta POP, SMTP, IMAP.
  • Filtros, firmas, PGP, un fierrazo!
  • Necesita librerias xforms 0.88.1 y xpm 3.4j ( la xpm viene en cualquier dist. decente; la xforms se baja en ftp://braggs.phys.uwm.edu/pub/xforms )
  • Ultima versión: 1.3
  • 9 pinguinos.
  • Home page: http://burka.netvision.net.il/xfmail/xfmail.html

Netscape Communicator

  • Que se puede decir que no se sepa?
  • El mejor navegador grafico de WWW.
  • Ultima versión: 4.6
  • 8 pinguinos y medio.
  • Home page: http://home.netscape.com

NcFTP

  • Excelente cliente de FTP.
  • Resume transferencias, tiene opcion de bookmarks.
  • Ultima versión: 3.0.0
  • 8 pinguinos
  • Home page: http://www.ncftp.com

Bladeenc

  • Codificador MP3.
  • Ultima versión: 0.76
  • 8 pinguinos.
  • Home page: http://home8.swipnet.se/~w-82625

xanim

  • Reproductor multimedia para Linux.
  • Pasa archivos AVI, MOV, FLI, FLC, WAV... y como 15 formatos mas...
  • Ultima versión: 2.80.1
  • 8 pinguinos y medio.
  • Home page: http://xanim.va.pubnix.com

cdda2wav

  • Ripper de CDs de audio.
  • Muy bueno, crea unos .wav de mucha calidad.
  • Ultima versión: 1.8

Corel Wordperfect para Linux

  • EL procesador de textos para linux
  • Genial, con todo lo que se puede esperar de un proc. de textos.
  • Ultima versión: 8
  • Comercial, pero se puede bajar una version gratuita para uso personal.
  • 8 pinguinos.
  • Home page: http://linux.corel.com

GIMP

  • Procesamiento de imagenes en Linux
  • Es GNU, no hay muchos programas gratuitos de procesamiento de imagenes.
  • Ultima versión: 1.0.5
  • Necesita librerias GTK ( http://www.gtk.org )
  • 8 pinguinos
  • Home page: http://www.gimp.org
